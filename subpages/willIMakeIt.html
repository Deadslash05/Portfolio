Will I Make It?

Project Status: Finished
Platforms: Itch.io
Duration: 4 months
Software Used: Unity
Languages Used: C#
Main Roles: Programmer

Will I Make It? is a rhythm game similar to games such as Wario Ware and Rhythm Heaven where you play as the astronaut Paco whose spaceship broke and tries to get home by playing minigames to fix it.

PLAY

My role

  • Coded half of all minigames: Fix Ship, Climb Ladder, Escape from Alien and Climb Mountain
  • Coded the system that changes the speed of the game when modifying the difficulty and applied it to all minigames
  • Coded the system that only allows you to input any key close to when a beat happens and applied it to all levels
